# zzoinn 랜딩페이지 수정 계획

## 변경사항 요약

### 1. 인트로 지구본 — 화면 꽉 채우는 실제 지구 모양
- 크기: `min(92vw, 92vh)` — 뷰포트를 거의 가득 채움
- 실제 대륙 윤곽 (아메리카, 유럽/아프리카, 아시아) SVG path로 픽토그램 표현
- `feTurbulence + feDisplacementMap` 필터로 손그림·귀여운 느낌 강화
- 기존 추상 블롭 → 인지 가능한 대륙 형태로 교체

### 2. 캐릭터가 걷는 반쪽 지구본 삽입
- `site-entered` 전환 후 상단에 반원(dome) SVG 표시
- 경도·위도 선, 대륙 흔적 그려 지구본임을 명확히
- 캐릭터(`walker-wrap`)가 이 반구 위에서 patrol 애니메이션

### 3. 왼쪽 탭 세로 나열
- 내비게이션: Magazine / Goods 링크를 `flex-direction: column`으로 쌓기
- 좌측 배치, 폰트 크기 및 간격 조정

### 4. 오른쪽 로고 — 집 모양 사각형 안에 zzo / inn
- "zzoinn" 을 "zzo" (위) + "inn" (아래) 2행으로 분리
- 컨테이너: 사각형 + 상단 삼각형 지붕 = CSS `clip-path` 집 모양
- 우측 배치, 강조 색상 배경
